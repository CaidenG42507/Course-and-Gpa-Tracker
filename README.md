# Course Gpa Tracker In Python

A simple GPA and course tracker that allows you to add and remove courses, calculate gpa, and save/load data.

# Simple Features
- Add a course (Gpa, name, credits)
- Remove a course
- Calculate GPA
- View all classes
- Save/Load files

# GPA Calculation
- Quality points (grade points * credits)
- GPA = Quality points / number of credits 

# How to run

1. Download this repo
2. Open terminal in the project folder
3. Run:

```bash
python Main.py
